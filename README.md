# ⚔️ dnd-forge

> Forge your hero. Manage your campaign. Roll with confidence.

**dnd-forge** is an open-source Python toolkit for creating and managing
Dungeons & Dragons 5e character sheets programmatically — with full
multiclass support, class-driven level-up logic, and JSON-based persistence.

---

## ✨ Features

- 🧙 Create D&D 5e characters with any class
- ⚔️ Full **multiclass support** — level up each class independently
- 📜 Class data loaded from **JSON templates** (easy to extend)
- 🎲 Automatic modifier, proficiency bonus and saving throw calculation
- 🏹 Skill proficiencies, armor & weapon competencies per class
- 💾 Serializable characters — save and load your party
- 🔌 Designed to be extended: subclasses, spells, feats coming soon
